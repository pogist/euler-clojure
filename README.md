## euler-clojure

#### Testing solutions

Each solution has an associated test suite to it e.g.:
- `src/euler/euler001.clj` -> `test/euler/euler001_test.clj`
- `src/euler/euler002.clj` -> `test/euler/euler002_test.clj`

And so on...

So you just need to run `lein test`.
